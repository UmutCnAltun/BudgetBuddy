# 💰 BudgetBuddy: Akıllı Kişisel Finans Yönetimi

Gelir ve giderlerinizi zahmetsizce takip etmeniz, harcama alışkanlıklarınızı analiz etmeniz ve finansal hedeflerinize ulaşmanız için geliştirilmiş, Python & Streamlit tabanlı bir bütçe yönetimi uygulamasıdır.

## 🌟 Öne Çıkan Özellikler

* Görsel Analiz Paneli: Harcamalarınızı kategorilerine göre ayırarak Pasta Grafik (Pie Chart) ve Sütun Grafikleri ile görselleştirin. Nereye ne kadar harcadığınızı bir bakışta görün.

* Dönemsel Takip: Harcamalarınızı aylık, haftalık veya özel tarih aralıklarına göre filtreleyerek geçmiş dönem performansınızı inceleyin.

* Veri Dışa Aktarma (Export): Kaydettiğiniz tüm verileri istediğiniz zaman CSV veya Excel formatında dışa aktararak bilgisayarınızda saklayın.

* Hızlı Veri Girişi: Gelir ve gider kalemlerini kategori, miktar ve tarih bilgisiyle saniyeler içinde kaydedin.

* Web Tabanlı Arayüz: Streamlit altyapısı sayesinde herhangi bir kurulum gerektirmeden tarayıcı üzerinden (veya yayındaki site üzerinden) erişim imkanı.

## 🛠️ Kullanılan Teknolojiler
* Python: Uygulamanın temel programlama dili.

* Streamlit: Hızlı ve etkileşimli web arayüzü geliştirme.

* Pandas: Güçlü veri işleme ve manipülasyon kütüphanesi.

* SQLite: Verilerin yerel olarak güvenle saklanması.

## 🚀 Hızlı Başlangıç
Projeyi yerel makinenizde çalıştırmak için:

Projeyi klonlayın:

'''
git clone https://github.com/UmutCnAltun/BudgetBuddy.git
'''

Sanal bir çalışma ortamı oluşturun ve aktif edin: 

'''
python3 -m venv .venv
.venv\\Scripts\\activate
'''

Gerekli kütüphaneleri yükleyin:

'''
pip install -r requirements.txt
'''

Uygulamayı başlatın:

'''
streamlit run app.py
'''
## 📊 Neden BudgetBuddy?

Bütçe yönetimi karmaşık tablolardan ibaret olmamalıdır. BudgetBuddy, karmaşık verileri anlamlı görsellere dönüştürerek finansal okuryazarlığınızı artırır ve bütçenizi kontrol altında tutmanızı sağlar.
